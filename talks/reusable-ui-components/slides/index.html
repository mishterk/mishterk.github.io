What do we mean by Reusable UI Components?

  • Packaged code that we can use across many projects with minimal configuration
  • HTML, CSS, JS, & PHP
  • Highly portable, ideally managed via a package manager
  • Preferably framework/CMS agnostic

Why do we care?

  • Redeveloping the same thing over and over is a pain
  • We can make more $$$ by building on our past efforts
  • Modular, packaged code makes for smoother living;
    • Easier to develop further
    • Easier to maintain
    • Easier to understand
